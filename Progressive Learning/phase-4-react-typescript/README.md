# React + TypeScript Track — starts week 3, 2x per week

**Goal:** Build enough React/TS fluency to contribute to frontend in a full-stack or security-tooling role.

## Progress checklist

### TypeScript fundamentals (Weeks 3–4)
- [ ] Types vs interfaces
- [ ] Union types, optional chaining
- [ ] Generics basics
- [ ] Type inference

### React fundamentals (Weeks 5–6)
- [ ] JSX, components, props
- [ ] useState, useEffect
- [ ] Event handling
- [ ] Conditional rendering, lists

### React intermediate (Weeks 7–8)
- [ ] useContext, useReducer
- [ ] Custom hooks
- [ ] Fetching data (fetch / axios)
- [ ] React Router basics

### Connect to Rails API (Week 9+)
- [ ] Build React frontend for the security dashboard portfolio project
- [ ] CORS configuration in Rails
- [ ] JWT authentication flow

## Security angle

TypeScript prevents a class of runtime errors that become security vulnerabilities —
undefined references, type coercion bugs, null pointer errors.
Frame your TS learning through this lens.
